Minimal Angular App
===================

This is a basic angular app using:

- ui.router
- ngResource
- angular-jwt

The front-end framework is `bootstrap#v4.0.0-alpha`.

It uses IIFE's to manage global variables.

## Setup 

To run the app, you need to run:

```bash
grunt
```
